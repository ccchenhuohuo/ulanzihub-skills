# 常见错误排查指南

本文档记录每次运行skill时遇到的问题及解决方案，随使用不断积累更新。

---

## StreamLoad 导入错误

### DATA_QUALITY_ERROR

| 项目 | 说明 |
|------|------|
| **错误信息** | StreamLoad返回 `Status: DataQualityError` |
| **常见原因** | JSON字段为空值、字段数量不匹配、格式错误 |
| **排查步骤** | 1. 检查JSON文件是否为空或字段缺失 |
| | 2. 验证 `convert_to_json.py` 的字段映射是否正确 |
| | 3. 用 `head -c 500 batch_0001.json` 检查JSON格式 |
| **修复方法** | 重新运行 `python3 convert_to_json.py --path {数据路径}` |

---

## CSV处理错误

### site字段为空

| 项目 | 说明 |
|------|------|
| **错误信息** | 导入Doris后site字段为NULL |
| **常见原因** | `merge_csv.py` 缺少site字段映射 |
| **排查步骤** | 1. 检查 `merged_data.csv` 是否有site列 |
| | 2. 核对 `merge_csv.py` 的 `col_map` |
| **修复方法** | 添加 `'site': 'site'` 到 `col_map` 并重新合并 |

### 列顺序不匹配

| 项目 | 说明 |
|------|------|
| **错误信息** | 导入后数据错位或报错 |
| **常见原因** | SELECT语句未指定列名，使用了 `SELECT *` |
| **排查步骤** | 1. 执行 `DESC flywheel.monthly_sales_with_dim` |
| | 2. 检查INSERT语句的列名顺序 |
| **修复方法** | 使用显式列名，不要用 `SELECT *` |

## JOIN错误

### 数据错乱

| 项目 | 说明 |
|------|------|
| **错误信息** | JOIN后数据与预期不符 |
| **常见原因** | 中间产物放错位置、文件被覆盖 |
| **排查步骤** | 1. 检查各阶段输出文件是否在正确目录 |
| | 2. 核对 `wc -l` 各阶段行数是否一致 |
| **修复方法** | 确保所有输出到用户指定的 `{数据路径}` |

### JOIN匹配率低

| 项目 | 说明 |
|------|------|
| **现象** | INNER JOIN后数据大量丢失 |
| **常见原因** | CN站点用product_id+year_num，US/JP/DE用sku_id+site+year_num |
| **排查步骤** | 检查 `references/join_rules.md` 确认JOIN条件 |
| **修复方法** | 按站点分别执行不同的JOIN条件 |

## 环境连接错误

### Doris MCP连接失败

| 项目 | 说明 |
|------|------|
| **错误信息** | MCP调用返回连接错误 |
| **常见原因** | Doris服务不可达、MCP配置问题 |
| **排查步骤** | 1. 确认Doris服务运行正常 |
| | 2. 检查MCP配置 |
| **修复方法** | 重启Doris MCP服务或联系管理员 |

### MySQL连接失败

| 项目 | 说明 |
|------|------|
| **错误信息** | `mysql: Can't connect to MySQL server` |
| **常见原因** | MySQL服务不可达、端口错误、密码错误 |
| **排查步骤** | 1. `telnet ${DORIS_HOST} ${MYSQL_PORT}` |
| | 2. 确认用户名密码正确 |
| **修复方法** | 检查网络连接或更新密码 |
