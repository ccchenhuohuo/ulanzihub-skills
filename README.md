# ulanzihub-skills

优篮子内部技能库 - Claude Code & OpenClaw 通用

## 什么是 ulanzihub-skills?

ulanzihub-skills 是优篮子公司的内部技能市场，提供 Claude Code 和 OpenClaw 兼容的技能。

## 快速开始

### 安装技能

```bash
# 使用 ulanzihub CLI
ulanzihub install company-code-review

# 或使用 npx
npx ulanzihub-skills install company-code-review
```

## 技能列表

### company-code-review

公司代码规范审查技能。按照公司代码规范审查代码，支持中文注释和命名规范。

### company-docs

公司文档助手。查询公司内部文档和流程，包括 HR、财务、研发流程等。

### feishu-bot

飞书机器人开发技能。开发飞书机器人和集成应用，支持消息卡片、交互组件等。
